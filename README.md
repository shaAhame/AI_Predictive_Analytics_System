# AI Business Forecasting Dashboard

A Streamlit app that predicts revenue, expense, and profit trends, and provides intelligent AI recommendations to mitigate losses.

## Features
- Forecasts future profit using Prophet
- Detects loss scenarios
- Generates business advice using AI (via OpenRouter API)

## Run the App
```bash
streamlit run dashboard/app.py
```
